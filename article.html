Les nouveautés ES2025 à connaître

📅 20 Mai 2025 ⏱ 5 min de lecture ✍️ BlogTech
JS

Introduction

Chaque année, le comité TC39 publie de nouvelles fonctionnalités pour JavaScript. ES2025 apporte son lot de nouveautés qui vont simplifier notre code au quotidien.

1. Iterator Helpers

Les itérateurs ont désormais des méthodes comme map(), filter() et take() directement disponibles sans avoir à tout convertir en tableau.

const result = [1, 2, 3, 4, 5].values() .filter(n => n % 2 === 0) .map(n => n * 10) .toArray(); // [20, 40]

2. Promise.try()

Une nouvelle méthode qui permet d'encapsuler n'importe quelle valeur ou fonction dans une promesse, qu'elle soit synchrone ou asynchrone.

Promise.try(() => fetchData()) .then(data => console.log(data)) .catch(err => console.error(err));

3. Explicit Resource Management

Le mot-clé using permet de gérer automatiquement la libération des ressources à la fin d'un bloc, similaire au using en C#.

{ using file = openFile('data.txt'); // Le fichier est automatiquement fermé ici }

Conclusion

ES2025 continue d'améliorer JavaScript avec des fonctionnalités pratiques. Restez à jour et testez ces nouveautés dans vos projets !
