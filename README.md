# Blog Tech T — Articles & Tutoriels

Un blog tech statique réalisé dans le cadre du projet GoMyCode Phase 1.

##  Technologies utilisées

- HTML5
- CSS3
- JavaScript (Vanilla)

## Fonctionnalités

- Recherche en temps réel dans les articles
- Filtres par catégorie (HTML, CSS, JavaScript, Outils)
- Mode sombre / clair avec sauvegarde
- Design responsive (mobile, tablette, desktop)
- Menu burger sur mobile
- Animations au scroll
- Page de détail d'article

##  Structure du projet

blog-tech/
├── index.html
├── style.css
├── script.js
└── article.html
